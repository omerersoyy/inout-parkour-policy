# inout-parkour-policy

Privacy policy page for **Inout Parkour** — a static HTML site hosted to satisfy app store requirements.

## Contents

- `index.html` — the privacy policy document for Inout Parkour (Ömer Ersoy, `com.innout.game`)

## Hosting

Deploy `index.html` as a publicly accessible URL (e.g. GitHub Pages) and link it in the Google Play /
App Store listings under the privacy policy field.

## Updating the policy

Edit `index.html` directly. Update the "Last updated" date at the top of the document whenever content
changes — especially once analytics or additional SDKs are added (see PHASE_10_ANALYTICS.md in the
game repo).
